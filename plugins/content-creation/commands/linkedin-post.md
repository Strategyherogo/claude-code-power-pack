# Skill: LinkedIn Post
Create engaging LinkedIn content.

## Auto-Trigger
**When:** "linkedin", "linkedin post", "professional post"

## Post Frameworks

### Framework 1: Story Hook
```
[Attention-grabbing first line]

[Set the scene - 1-2 sentences]

[The challenge/conflict]

[The turning point]

[The lesson/insight]

[Call to engage: question or takeaway]
```

### Framework 2: Contrarian Take
```
Unpopular opinion: [contrarian statement]

Here's why:

1. [Reason 1]
2. [Reason 2]
3. [Reason 3]

The real issue is [deeper insight]

What do you think? [Question]
```

### Framework 3: Listicle
```
[X] things I learned about [topic]:

1. [Lesson] - [brief explanation]

2. [Lesson] - [brief explanation]

3. [Lesson] - [brief explanation]

4. [Lesson] - [brief explanation]

5. [Lesson] - [brief explanation]

Which one resonates most?
```

### Framework 4: Before/After
```
1 year ago: [past state]

Today: [current state]

What changed:

→ [Change 1]
→ [Change 2]
→ [Change 3]

The biggest shift was [key insight]

[CTA or question]
```

## LinkedIn Best Practices
```
✅ Do:
- Hook in first line (it gets truncated)
- Use line breaks for readability
- Include a clear takeaway
- End with engagement prompt
- Post consistently (2-3x/week)
- Engage in comments within 1 hour

❌ Don't:
- Start with "I'm excited to announce..."
- Use too many hashtags (3-5 max)
- Post external links in main post (comment)
- Be overly promotional
- Ignore comments
- Post and ghost
```

## Post Templates

### Achievement Post
```
Just hit a milestone: [achievement]

But here's what nobody tells you about [topic]:

[Insight 1]
[Insight 2]
[Insight 3]

The real win wasn't [obvious thing].

It was [deeper lesson].

What milestones are you working toward?
```

### Value Post
```
Stop [common mistake].

Start [better approach] instead.

Here's the difference:

❌ [Wrong way]:
- [Problem 1]
- [Problem 2]

✅ [Right way]:
- [Benefit 1]
- [Benefit 2]

I made this shift [timeframe] ago.

Results: [specific outcome]

Try it this week and let me know how it goes.
```

### Question Post
```
I've been thinking about [topic] lately.

[Context - 2-3 sentences]

My take: [your perspective]

But I'm curious:

[Specific question to audience]

Drop your thoughts below 👇
```

## Formatting Tips
```
Line breaks: After every 1-2 sentences
Emojis: Sparingly, for visual breaks
Hashtags: At the end, 3-5 relevant ones
Length: 1200-1500 characters optimal
Hook: First 150 characters are crucial
```

## Hashtag Strategy
```
Mix of:
- 1 broad (#Leadership, #Marketing)
- 2-3 medium (#StartupLife, #ProductManagement)
- 1-2 niche (#SaaS, #B2BSales)

For your niche:
#AI #Automation #TechConsulting #ADHD
#ProductManagement #IndieHacker
```

## Engagement Strategy
```
After posting:
- Respond to every comment in first hour
- Ask follow-up questions
- Thank people for sharing
- Pin best comment

On others' posts:
- Add value, don't just agree
- Share your experience
- Ask thoughtful questions
- Build relationships
```
