# Skill: c2c:blog
Full blog post creation workflow.

## Auto-Trigger
**When:** "write blog", "blog post", "article", "long-form"

## Blog Creation Workflow

### Phase 1: Research (10 min)
```
□ Keyword research (volume, difficulty)
□ Competitor analysis (top 3 articles)
□ Angle identification (what's missing)
□ Source gathering (stats, quotes)
```

### Phase 2: Outline (5 min)
```markdown
## [Title]

### Introduction
- Hook: [specific hook]
- Context: [why this matters now]
- Promise: [what reader will learn]

### Section 1: [Topic]
- Main point
- Evidence/example
- Transition

### Section 2: [Topic]
- Main point
- Evidence/example
- Transition

### Section 3: [Topic]
- Main point
- Evidence/example
- Transition

### Conclusion
- Summary
- Key takeaway
- CTA
```

### Phase 3: Draft (30 min)
```
Write following these rules:
- Hook readers in first 50 words
- One idea per paragraph
- Use subheadings every 200-300 words
- Include examples and specifics
- Write conversationally
```

### Phase 4: Edit (15 min)
```
□ Cut unnecessary words
□ Strengthen opening
□ Check flow between sections
□ Verify facts and links
□ Add internal links
□ Optimize for target keyword
```

### Phase 5: Polish (10 min)
```
□ Write meta description
□ Select featured image
□ Add alt text to images
□ Format for readability
□ Final proofread
```

## Blog Post Structure

### Standard Post (1500 words)
```
Introduction: 100-150 words
Body: 1000-1200 words (3-4 sections)
Conclusion: 100-150 words
```

### Long-Form Guide (3000+ words)
```
Introduction: 200-300 words
Body: 2500-3000 words (5-7 sections)
Conclusion: 200-300 words
```

### Quick Post (800 words)
```
Introduction: 50-75 words
Body: 600-700 words (2-3 sections)
Conclusion: 50-75 words
```

## SEO Checklist
```
□ Target keyword in title
□ Target keyword in first 100 words
□ Target keyword in H2s
□ Related keywords throughout
□ Meta description (150-160 chars)
□ URL slug optimized
□ Internal links (2-3)
□ External links (1-2 authoritative)
□ Image alt text optimized
```

## Output
Save to: `content/blog/YYYY-MM-DD-slug.md`
