## Newsletter Best Practices
```
✅ Do:
- Consistent send day/time
- Scannable format
- One clear CTA
- Personal tone
- Provide value every issue
- Respect inbox

❌ Don't:
- Wall of text
- Multiple competing CTAs
- Only promotional content
- Inconsistent schedule
- Forget mobile readers
```

## Subject Line Formulas
```
- [Number] [Things] [Outcome]: "5 Tools That Save Me Hours"
- How I [Achievement]: "How I Grew to 10K Subscribers"
- [Time] [Topic] Update: "Weekly AI Update"
- The [Adjective] [Thing]: "The Underrated Skill"
- [Question]?: "Are You Making This Mistake?"
```

## Metrics to Track
```
Open rate: >40% good
Click rate: >3% good
Reply rate: Track for engagement
Unsubscribe: <0.5% acceptable
Growth rate: Track month over month
```
