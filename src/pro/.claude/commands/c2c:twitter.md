# Skill: c2c:twitter
Create engaging Twitter/X content.

## Auto-Trigger
**When:** "tweet", "twitter", "x post", "thread"

## Tweet Formats

### Single Tweet (280 chars)
```
[Hook - first 100 chars most important]

[Main point]

[CTA or question]
```

### Thread Structure
```
Tweet 1: Hook (promise what thread delivers)
Tweet 2: Context/setup
Tweet 3-7: Main points (one per tweet)
Tweet 8: Summary
Tweet 9: CTA + retweet ask
```

## Thread Templates

### Story Thread
```
1/ [Provocative hook]

This is the story of [what happened].

Here's how it went: 🧵

2/ It started when [setup]

3/ Then [conflict/challenge]

4/ I tried [first attempt - failed]

5/ What I learned: [insight]

6/ So I [new approach]

7/ Result: [outcome]

8/ The lesson: [takeaway]

9/ If this was useful, RT the first tweet.

Follow me @handle for more [topic].
```

```

```

## Tweet Best Practices
```
✅ Do:
- Start with hook
- Use line breaks
- Numbers work (78% better)
- Ask questions
- Tag relevant people
- Use 1-2 relevant hashtags

❌ Don't:
- Start with "I"
- Use too many hashtags
- Post links in main tweet (put in reply)
- Write walls of text
- Be boring
```

## Engagement Tactics
```
□ Post during peak hours (9-11am, 7-9pm)
□ Reply to comments within first hour
□ Quote tweet with added value
□ Engage with others before/after posting
□ Pin best performing tweet
```

## Content Buckets
```
Personal: Stories, lessons, behind-the-scenes
Educational: How-tos, tips, frameworks
Curated: Quotes, resources, tools
Engagement: Questions, polls, hot takes
Promotional: 10% max (product, newsletter)
```
