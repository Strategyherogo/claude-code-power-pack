# Skill: Design Website
Website design and structure planning.

## Auto-Trigger
**When:** "website", "landing page", "homepage", "web design"

## Design Framework

### 1. Define Purpose
```
□ Primary goal (conversions, info, portfolio)
□ Target audience
□ Key messages (max 3)
□ Call to action
□ Success metrics
```

### 2. Structure
```markdown
## Homepage Structure

### Above the Fold
- Hero headline (6-12 words)
- Subheadline (15-25 words)
- Primary CTA button
- Hero image/video
- Social proof snippet

### Section 2: Problem/Solution
- Pain point statement
- Your solution
- Key benefits (3)

### Section 3: Features/How It Works
- 3-4 feature blocks
- Icons + short descriptions
- Optional screenshots

### Section 4: Social Proof
- Testimonials (2-3)
- Logos of clients
- Stats/metrics

### Section 5: Pricing (if applicable)
- Pricing tiers
- Feature comparison
- FAQ

### Footer
- Navigation links
- Contact info
- Legal links
- Newsletter signup
```

### 3. Copy Guidelines
```
Headlines: Benefit-focused, specific
Subheads: Expand on headline
Body: Short paragraphs (2-3 sentences)
CTAs: Action verbs ("Get Started", "Try Free")
```

## Tech Stack Recommendations

### Static Sites (Simple)
```
- Astro + Tailwind
- Next.js static export
- Hugo + custom theme
```

### Dynamic Sites (Complex)
```
- Next.js + Vercel
- Remix + Cloudflare
- SvelteKit
```

### No-Code Options
```
- Framer
- Webflow
- Carrd (simple)
```
