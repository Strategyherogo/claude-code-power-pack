# Skill: c2c:quick-blog
Rapid blog post in under 15 minutes.

## Auto-Trigger
**When:** "quick blog", "fast post", "quick article"

## Quick Blog Formula

### Time Budget: 15 minutes
```
Research: 3 min (skip if topic known)
Outline: 2 min
Draft: 8 min
Edit: 2 min
```

### Structure: 5-Paragraph Format
```markdown
## [Title with Keyword]

**Paragraph 1: Hook + Promise**
[Attention-grabbing opener. What reader will learn.]

**Paragraph 2: Context/Problem**
[Why this matters. The challenge being addressed.]

**Paragraph 3: Solution/Main Point**
[The answer. The key insight. The how-to.]

**Paragraph 4: Evidence/Example**
[Proof it works. Specific example or case.]

**Paragraph 5: Conclusion + CTA**
[Summary. What to do next. Call to action.]
```

### Quick Templates

#### How-To (5 steps)
```markdown
# How to [Achieve X] in [Timeframe]

Want to [desired outcome]? Here's a quick way to get there.

## The Problem
[Brief pain point]

## The Solution
1. **[Step 1]** - [one sentence]
2. **[Step 2]** - [one sentence]
3. **[Step 3]** - [one sentence]
4. **[Step 4]** - [one sentence]
5. **[Step 5]** - [one sentence]

## Quick Example
[Brief real example]

## Your Turn
[CTA]
```

#### Listicle (5 items)
```markdown
# [X] [Things/Ways/Tips] for [Audience/Goal]

[One sentence intro]

## 1. [Item]
[2-3 sentences]

## 2. [Item]
[2-3 sentences]

## 3. [Item]
[2-3 sentences]

## 4. [Item]
[2-3 sentences]

## 5. [Item]
[2-3 sentences]

## The Bottom Line
[Summary + CTA]
```

#### Opinion/Take
```markdown
# Why [Contrarian Opinion]

[Provocative opener]

## The Common Belief
[What most people think]

## The Reality
[Your counter-argument]

## What This Means
[Implications]

## My Recommendation
[What to do instead]
```

### Speed Writing Tips
```
✅ Use templates
✅ Write ugly first draft
✅ Don't research mid-write
✅ Use placeholder [STAT] tags
✅ Edit in second pass
✅ Set a timer
```

### Quick Quality Check
```
□ Title is clear and specific
□ Opening hooks attention
□ One clear takeaway
□ Has a CTA
□ No glaring typos
```
