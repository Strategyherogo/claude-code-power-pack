**What you'll learn:** [outcome]
**Time required:** [estimate]
**Prerequisites:** [what they need to know/have]
**Difficulty:** Beginner / Intermediate / Advanced
```

```markdown

- [Prerequisite 1]

## Tutorial Best Practices
```
✅ Do:
- Show expected output after each step
- Include checkpoints
- Anticipate errors
- Use copy-pasteable code
- Test every step yourself
- Include screenshots

❌ Don't:
- Skip "obvious" steps
- Use unexplained jargon
- Assume prior knowledge
- Leave code untested
- Forget error handling
```

## Tutorial Types

### Quick Tutorial (10 min)
```
- 3-5 steps
- Single concept
- Minimal setup
- Immediate result
```

### Standard Tutorial (30 min)
```
- 7-10 steps
- Build something real
- Include configuration
- Handle common errors
```

### Deep Dive (60+ min)
```
- 15+ steps
- Production-ready result
- Best practices included
- Multiple alternatives shown
```
