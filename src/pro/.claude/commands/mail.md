# Skill: mail
Email composition and management.

## Auto-Trigger
**When:** "email", "send email", "compose email", "draft email"

## Email Templates

### Professional Outreach
```
Subject: [Specific, benefit-focused subject]

Hi [Name],

[Hook - why you're reaching out]

[Context - brief background]

[Ask - what you want]

[Close - next step]

Best,
[Your name]

P.S. [Optional additional hook]
```

### Follow-Up
```
Subject: Re: [Previous subject] / Following up on [topic]

Hi [Name],

Wanted to follow up on [previous topic/meeting].

[Brief reminder of context]

[Question or ask]

Let me know if you have any questions.

Best,
[Name]
```

### Introduction Request
```
Subject: Intro request: [You] <> [Target]

Hi [Connector],

Hope you're doing well!

I noticed you're connected with [Target Name] at [Company]. I'd love an introduction if you think it makes sense.

**Quick context:**
[1-2 sentences on why you want to connect]

**What I can offer them:**
[Value you bring]

Happy to send over a forwardable blurb if helpful.

Thanks!
[Name]
```

### Meeting Request
```
Subject: Quick chat about [topic]?

Hi [Name],

[1 sentence context/reason for meeting]

Would you have 15-30 minutes for a quick call this week?

I'm generally free [suggest 2-3 times].

If easier, here's my Calendly: [link]

Thanks,
[Name]
```

### Thank You
```
Subject: Thank you for [specific thing]

Hi [Name],

Just wanted to say thank you for [specific thing they did].

[Why it mattered / impact]

Really appreciate it.

Best,
[Name]
```

### Apology / Issue Response
```
Subject: Apology re: [issue]

Hi [Name],

I apologize for [specific issue].

**What happened:**
[Brief explanation - no excuses]

**What we're doing:**
[Steps to fix / prevent recurrence]

**Next steps:**
[What they can expect]

I take full responsibility for this. Please let me know if there's anything else I can do.

Sincerely,
[Name]
```

```
**In Progress:**
- [Current work]

- [Upcoming action]

- [If any]

[Name]
```

## Email Best Practices
```
✅ Do:
- Subject line: specific, actionable
- One topic per email
- Clear ask/CTA
- Keep it short (5 sentences max)
- Proofread before sending
- Mobile-friendly formatting

❌ Don't:
- Vague subjects ("Quick question")
- Multiple asks in one email
- Wall of text
- Reply all unnecessarily
- Send emotional emails immediately
- CC people who don't need to be
```

## Subject Line Formulas
```
Request: "Quick question about [topic]"
Update: "[Project] Update - [Date]"
Follow-up: "Following up: [original topic]"
Introduction: "Intro: [Name] <> [Name]"
Meeting: "[Meeting type] - [Topic] - [Date]"
Feedback: "Feedback on [item]"
Thank you: "Thank you for [specific thing]"
```

## Email Checklist
```
□ Subject line is specific
□ Correct recipient(s)
□ Greeting appropriate
□ Purpose clear in first sentence
□ Ask is specific and clear
□ Appropriate sign-off
□ Attachments attached
□ Links work
□ Proofread for typos
□ Tone check (read aloud)
```
