# Skill: Case Study
Create compelling customer case studies.

## Auto-Trigger
**When:** "case study", "customer story", "success story", "testimonial"

## Case Study Structure

### 1. Executive Summary (50 words)
```
[Company] faced [challenge]. Using [solution], they achieved [result 1], [result 2], and [result 3].
```

### 2. Customer Profile
```markdown
**Company:** [Name]
**Industry:** [Industry]
**Size:** [employees/revenue]
**Location:** [HQ location]
**Use Case:** [What they use product for]
```

### 3. The Challenge
```markdown
## The Challenge

[Customer] was experiencing [problem description].

### Key Pain Points
- [Pain point 1 with specific impact]
- [Pain point 2 with specific impact]
- [Pain point 3 with specific impact]

### The Stakes
[What would happen if they didn't solve this]

> "[Direct quote from customer about the problem]"
> — [Name], [Title]
```

### 4. The Solution
```markdown
## The Solution

[Customer] chose [Product] because [key decision factors].

### Implementation
[Brief description of how they implemented]

### Key Features Used
- [Feature 1] - [how they use it]
- [Feature 2] - [how they use it]
- [Feature 3] - [how they use it]

> "[Quote about why they chose this solution]"
```

### 5. The Results
```markdown
## The Results

### By the Numbers
| Metric | Before | After | Change |
|--------|--------|-------|--------|
| [Metric 1] | [X] | [Y] | +Z% |
| [Metric 2] | [X] | [Y] | +Z% |
| [Metric 3] | [X] | [Y] | +Z% |

### Qualitative Wins
- [Benefit 1 - with specifics]
- [Benefit 2 - with specifics]
- [Benefit 3 - with specifics]

> "[Quote about results]"
```

### 6. Looking Ahead
```markdown
## What's Next

[Customer]'s plans with [Product] going forward.

> "[Quote about future/recommendation]"
```

## Interview Questions Template
```
About the Challenge:
- What was happening before [Product]?
- How was this affecting your team/business?
- What had you tried before?

About the Decision:
- How did you find [Product]?
- What made you choose us over alternatives?
- What were your concerns before buying?

About Implementation:
- How was the setup process?
- How long until you saw results?
- What surprised you?

About Results:
- What specific results have you seen?
- How has this impacted your team?
- Would you recommend us? Why?
```

## Case Study Formats

### Long-Form (PDF, 2-4 pages)
```
Best for: Sales collateral, website
Includes: Full narrative, graphics, quotes
```

### Short-Form (1 page)
```
Best for: Quick reference, emails
Includes: Summary, key stats, one quote
```

### Video (2-3 minutes)
```
Best for: Website, social media
Includes: Interview clips, B-roll, results overlay
```

### Slide Version (5-7 slides)
```
Best for: Sales presentations
Includes: One key point per slide
```

## Design Elements
```
□ Customer logo (with permission)
□ Key metric callouts
□ Pull quotes styled prominently
□ Before/after visuals
□ Screenshots if relevant
□ CTA at the end
```
