# Skill: Demo Script
Create effective product demo scripts.

## Auto-Trigger
**When:** "demo", "walkthrough", "product tour", "show how"

## Demo Structure: SHOW Framework

### S - Setup Context (30 seconds)
```
"Before I show you [Product], let me quickly understand...
- What's your main goal with [area]?
- What are you using today?
- What's the biggest pain point?"

[Tailor demo based on answers]
```

### H - Hook with Outcome (1 minute)
```
"Let me show you how [similar customer] went from [pain] to [result].

By the end of this demo, you'll see:
1. How to [key outcome 1]
2. How to [key outcome 2]
3. How to [key outcome 3]"
```

### O - Overview + Walkthrough (5-7 minutes)
```
"Here's how it works..."

[Screen 1: First impression]
- This is your dashboard
- Point out key navigation
- Show primary action path

[Screen 2: Core feature]
- This solves [pain point 1]
- Here's how: [action]
- Result: [outcome]

[Screen 3: Second feature]
- This solves [pain point 2]
- Here's how: [action]
- Result: [outcome]

[Screen 4: Differentiator]
- Most tools can't do this
- Our unique approach
- Why it matters to you
```

### W - Wrap with Value (2 minutes)
```
"So to recap:
- [Outcome 1] - which means [benefit]
- [Outcome 2] - which means [benefit]
- [Outcome 3] - which means [benefit]

For teams like yours, this typically means [big picture result]."
```

## Demo Script Template
```markdown
## Demo Script: [Product Name]

### Pre-Demo Setup
- [ ] Demo environment ready
- [ ] Sample data loaded
- [ ] Backup plan if tech fails
- [ ] Notes on prospect's situation

### Opening (30s)
"Hi [Name], thanks for joining. Before we dive in, I'd love to understand:
- [Discovery question 1]
- [Discovery question 2]

Great. Let me show you exactly how we can help with that."

### Hook (1min)
"[Customer name] was dealing with [similar challenge]. Using [Product], they [specific result] in [timeframe].

Let me show you how..."

### Walkthrough (5-7min)

**Screen 1: Dashboard**
[Screenshot/description]
"This is the first thing you see. Notice [key element]. This gives you [benefit]."

**Screen 2: Core Feature**
[Screenshot/description]
"Here's where the magic happens. Let me show you..."
[Demo the action]
"See how that [result]? That's [X hours] saved every [time period]."

**Screen 3: Secondary Feature**
[Screenshot/description]
"Another thing that helps with [pain point]..."

**Screen 4: Integration/Setup**
"Getting started is [easy/supported]. [Brief setup overview]."

### Objection Handling
"You might be wondering about [common concern]. Here's how we handle that..."

### Close (2min)
"Based on what you've seen:
- Does this address [their stated goal]?
- What questions do you have?
- Would it make sense to [next step]?"

### Next Steps
- [ ] Trial setup
- [ ] Send follow-up email
- [ ] Schedule implementation call
```

## Demo Best Practices
```
✅ Do:
- Ask questions first
- Use their language
- Show, don't tell
- Focus on outcomes
- Pause for questions
- Watch for reactions

❌ Don't:
- Feature dump
- Rush through
- Ignore questions
- Use jargon
- Go over time
- Forget the ask
```

## Backup Plans
```
If internet fails: Pre-recorded video
If data is wrong: "Let me show you in sandbox"
If feature doesn't work: "Here's how it normally works..."
If they're confused: "Let me step back and explain..."
```
