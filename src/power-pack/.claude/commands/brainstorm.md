# Skill: Brainstorm
Generate and explore ideas systematically.

## Auto-Trigger
**When:** "brainstorm", "ideas", "what if", "explore options", "creative"

## Brainstorm Framework

### Phase 1: Diverge (Generate)
```
Rules:
✅ Quantity over quality
✅ No criticism allowed
✅ Wild ideas welcome
✅ Build on others' ideas
✅ One idea at a time

Goal: 20+ ideas in 10 minutes
```

### Phase 2: Cluster (Organize)
```
Group ideas by:
□ Theme/category
□ Effort required
□ Impact potential
□ Novelty level
□ Dependencies
```

### Phase 3: Converge (Select)
```
Evaluate using:
□ Feasibility (can we do it?)
□ Desirability (do users want it?)
□ Viability (does it make sense?)
□ Alignment (fits our goals?)
```

## Brainstorm Prompts

### For Features
```
- What would delight users?
- What's the competitor doing?
- What's the lazy solution?
- What's the expensive solution?
- What would 10x this?
- What's the opposite approach?
```

### For Problems
```
- What's the root cause?
- Who else has this problem?
- What if we didn't solve it?
- What would make it worse?
- What worked before?
- What's the constraint we're assuming?
```

### For Names/Branding
```
- What does it do? (verb)
- What is it? (noun)
- How does it feel? (adjective)
- What's the metaphor?
- What's the opposite?
- What's the pun?
```

## Brainstorm Techniques
- **SCAMPER:** Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse
- **Six Hats:** Facts, Emotions, Caution, Benefits, Creativity, Process
- **5 Whys:** Keep asking "why" to find root
- **Worst Idea:** Generate terrible ideas, then invert
