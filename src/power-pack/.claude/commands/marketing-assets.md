# Skill: Marketing Assets
Create marketing materials and collateral.

## Auto-Trigger
**When:** "marketing", "assets", "collateral", "promotional"

## Asset Types

### Digital Assets
```
□ Social media graphics (1080x1080, 1200x628)
□ Email headers (600x200)
□ Banner ads (various sizes)
□ Blog post images
□ Presentation slides
□ Infographics
```

### Document Assets
```
□ One-pagers
□ Case studies
□ White papers
□ Product sheets
□ Pitch decks
□ Proposals
```

### Video Assets
```
□ Explainer videos
□ Demo videos
□ Testimonial clips
□ Social clips (15-60s)
□ Webinar recordings
```

## Asset Creation Workflow

### 1. Brief Definition
```markdown
**Asset:** [type]
**Purpose:** [goal]
**Audience:** [target]
**Key Message:** [main point]
**CTA:** [desired action]
**Deadline:** [date]
```

### 2. Content Gathering
```
□ Copy/messaging approved
□ Brand assets available (logos, colors, fonts)
□ Images/photos selected
□ Data/stats verified
□ Testimonials/quotes collected
```

### 3. Creation
```
□ Draft version
□ Internal review
□ Revisions
□ Final approval
□ Export in required formats
```

### 4. Distribution
```
□ Upload to asset library
□ Share with team
□ Schedule for campaigns
□ Track usage/performance
```

## Asset Specifications

### Social Media Sizes
```
Instagram Post: 1080 x 1080
Instagram Story: 1080 x 1920
LinkedIn Post: 1200 x 628
Twitter Post: 1200 x 675
Facebook Cover: 820 x 312
```

### Document Specs
```
One-pager: 1 page, PDF
Case Study: 2-4 pages, PDF
Pitch Deck: 10-15 slides, PPTX/PDF
White Paper: 5-15 pages, PDF
```

## Brand Consistency Checklist
```
□ Correct logo version used
□ Brand colors applied
□ Approved fonts used
□ Tone of voice consistent
□ Legal disclaimers included
□ Contact info current
```
