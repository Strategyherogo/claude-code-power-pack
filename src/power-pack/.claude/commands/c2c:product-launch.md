# Skill: c2c:product-launch
Create product launch content package.

## Auto-Trigger
**When:** "launch", "product launch", "release", "announcement"

## Launch Content Package

### 1. Product Hunt Listing
```markdown
## Tagline (60 chars max)
[Benefit-focused, specific]

## Description (260 chars)
[What it does] + [Who it's for] + [Key differentiator]

## First Comment (Maker's Comment)
Hey PH! 👋

I'm [name], maker of [product].

**The problem:** [pain point]

**The solution:** [what we built]

**What makes it different:**
• [Differentiator 1]
• [Differentiator 2]
• [Differentiator 3]

We're offering [special deal] for the PH community.

Would love your feedback! 🙏

## Gallery
- Hero image (1270x760)
- Feature screenshots (3-5)
- Demo GIF
```

## Launch Checklist
```
Pre-Launch:
□ Landing page live
□ Payment processing tested
□ Support system ready
□ Analytics tracking
□ Email sequences set

Launch Day:
□ Product Hunt submit (12:01 AM PT)
□ Tweet sent
□ Email sent to list
□ LinkedIn posted
□ Communities notified
□ Friends asked to support

Post-Launch:
□ Respond to all comments
□ Thank supporters
□ Monitor for issues
□ Collect feedback
□ Document learnings
```
