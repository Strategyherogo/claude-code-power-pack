# Skill: focus
Enter focused work mode with clear objectives.

## Auto-Trigger
**When:** "focus", "deep work", "concentrate", "no distractions"

## Focus Session Setup

### Define the Session
```markdown
## Focus Session: [Goal]
**Duration:** [time] (e.g., 90 min)
**Start:** [time]
**End:** [time]

### Single Objective
[One clear, specific outcome]

### Success Criteria
When this session ends, I will have:
- [ ] [specific deliverable]

### Not Doing (Park These)
- [distraction 1]
- [distraction 2]
- [other task]
```

### Pre-Focus Checklist
```
□ Clear single objective defined
□ End time set
□ Notifications silenced
□ Water/coffee ready
□ Bathroom break taken
□ Open tabs closed (except needed)
□ Phone away
□ "Do not disturb" on
```

## Focus Protocols

### Pomodoro (25/5)
```
25 min work → 5 min break → repeat
After 4 cycles: 15-30 min long break
```

### Deep Work Block (90 min)
```
90 min uninterrupted work
15-20 min break
Maximum 2-3 blocks per day
```

### Time Boxing
```
Set hard deadline for task
When time's up, stop (even if incomplete)
Review what got done
Adjust estimate for next time
```

## During Focus

### If Distracted
```
1. Notice the distraction
2. Write it down (park it)
3. Return to task immediately
4. Don't judge yourself
```

### If Stuck
```
1. Define the exact blocker
2. Try for 5 more minutes
3. If still stuck, write down the problem
4. Move to different part of task
5. Come back to blocker after break
```

### Quick Capture
Keep notepad for capturing:
```
- Ideas that pop up
- Tasks to do later
- Questions to research
- People to contact
```

## End of Focus Session

### Session Review
```markdown
### Focus Session Complete: [Goal]

**Planned:** [original objective]
**Actual:** [what was accomplished]

**Completion:** [X]%

**What worked:**
- [observation]

**What didn't:**
- [observation]

**Carry forward:**
- [ ] [remaining task]

**Energy level:** [1-10]
```

## Focus Environment
```
Physical:
□ Clean desk
□ Good lighting
□ Comfortable temperature
□ Noise managed (music/silence/white noise)

Digital:
□ Single browser window
□ Only necessary apps open
□ Slack/email closed
□ Phone on silent + away
```

## Recovery Between Sessions
```
- Stand up and move
- Look away from screen (20-20-20 rule)
- Hydrate
- Brief walk if possible
- Light snack if needed
- No social media/news
```
