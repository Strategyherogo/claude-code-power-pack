# Skill: Landing Copy
Write high-converting landing page copy.

## Auto-Trigger
**When:** "landing page copy", "website copy", "conversion copy", "sales page"

## Copy Framework: PAS + AIDA Hybrid

### Above the Fold
```markdown
## Headline (Problem-Agitate or Benefit)
[6-12 words, specific benefit or pain point addressed]

## Subheadline (Solution hint)
[15-25 words expanding on headline]

## CTA Button
[Action verb + value: "Get Started Free" / "See How It Works"]
```

### Section: Problem
```markdown
## Sound Familiar?
- [Pain point 1 - specific scenario]
- [Pain point 2 - emotional impact]
- [Pain point 3 - cost/consequence]

You're not alone. [Stat about how common this is].
```

### Section: Solution
```markdown
## Introducing [Product Name]
[One sentence describing what it is]

[2-3 sentences on how it solves the problem differently]

### How It Works
1. **[Step 1]** - [Brief description]
2. **[Step 2]** - [Brief description]
3. **[Step 3]** - [Brief description]
```

### Section: Benefits
```markdown
## Why [Product Name]?

### [Benefit 1 - Specific outcome]
[2-3 sentences with proof/specifics]

### [Benefit 2 - Emotional benefit]
[2-3 sentences with proof/specifics]

### [Benefit 3 - Differentiator]
[2-3 sentences with proof/specifics]
```

### Section: Social Proof
```markdown
## Trusted By [Number] [Type of Users]

> "[Testimonial quote - specific result]"
> — [Name], [Title], [Company]

### Results Our Customers See
- [Metric 1]: [Result]
- [Metric 2]: [Result]
- [Metric 3]: [Result]
```

### Section: Pricing
```markdown
## Simple, Transparent Pricing

### [Plan Name]
$XX/month
- [Feature 1]
- [Feature 2]
- [Feature 3]

[CTA: Start Free Trial]
```

### Section: FAQ
```markdown
## Frequently Asked Questions

**Q: [Common objection as question]**
A: [Address objection directly]

**Q: [Implementation concern]**
A: [Reassure with specifics]

**Q: [Pricing/value question]**
A: [Justify value]
```

### Final CTA
```markdown
## Ready to [Desired Outcome]?
[Urgency or guarantee statement]

[Primary CTA Button]

[Secondary option: "Or schedule a demo"]
```

## Copy Checklist
```
□ Headline is specific and benefit-focused
□ Subheadline adds context
□ CTAs are action-oriented
□ Benefits > Features
□ Social proof is specific
□ Objections are addressed
□ Mobile-friendly (short paragraphs)
□ One clear CTA per section
```

## Power Words
```
Free, New, Proven, Easy, Fast, Simple
Guaranteed, Exclusive, Limited, Instant
Save, Discover, Learn, Get, Start
```
