Reflect on work and extract learnings.

## Auto-Trigger
**When:** "reflect", "what did I learn", "retrospective", "lessons"

## Reflection Prompts
```
When stuck:
- What's the real problem here?
- What am I assuming?
- What would I tell a friend?

When successful:
- What specifically worked?
- Can I replicate this?
- Who helped make this happen?

When failed:
- What can I learn from this?
- What would I do differently?
- Is this a pattern?
```

## Action: After Reflection
```
1. Extract 1-3 actionable insights
2. Add to personal knowledge base
3. Schedule any follow-up actions
4. Share learnings if relevant
```
