# Skill: Content Brief
Create detailed content briefs for articles/posts.

## Auto-Trigger
**When:** "content brief", "blog brief", "article brief", "write brief"

## Content Brief Template

### Overview
```markdown
## Content Brief: [Working Title]

**Type:** Blog Post / Article / Guide / Tutorial
**Target Length:** [word count]
**Target Publish:** [date]
**Author:** [name]
**Reviewer:** [name]
```

### Audience & Intent
```markdown
### Target Audience
**Primary:** [specific persona]
**Secondary:** [if applicable]

### Search Intent
**Query:** [target keyword/phrase]
**Intent Type:** Informational / Commercial / Transactional
**Funnel Stage:** Awareness / Consideration / Decision

### Reader's Situation
- They're trying to: [goal]
- They're struggling with: [pain]
- After reading, they should: [outcome]
```

### Content Requirements
```markdown
### Key Message
[One sentence - what's the main takeaway?]

### Supporting Points (must include)
1. [Point 1]
2. [Point 2]
3. [Point 3]

### Questions to Answer
- [Question reader is asking 1]
- [Question reader is asking 2]
- [Question reader is asking 3]

### Must Mention
- [Required topic/feature/product]
- [Required CTA]
- [Required link]

### Avoid
- [Topic to stay away from]
- [Competitors not to mention]
- [Outdated information]
```

### SEO Requirements
```markdown
### Keywords
**Primary:** [main keyword] - Volume: [X] - Difficulty: [Y]
**Secondary:** [keyword 2], [keyword 3]
**Long-tail:** [phrase 1], [phrase 2]

### On-Page SEO
- Title tag: [suggestion]
- Meta description: [suggestion]
- URL slug: [suggestion]
- H2s should include: [keywords]

### Internal Links
- Link to: [page 1]
- Link to: [page 2]

### External Links
- Reference: [source 1]
- Reference: [source 2]
```

### Outline
```markdown
### Proposed Outline

**Introduction** (100 words)
- Hook: [specific hook]
- Context: [why this matters]
- Thesis: [main argument]

**Section 1: [Title]** (300 words)
- Point: [main point]
- Evidence: [support]
- Example: [specific example]

**Section 2: [Title]** (300 words)
- Point: [main point]
- Evidence: [support]
- Example: [specific example]

**Section 3: [Title]** (300 words)
- Point: [main point]
- Evidence: [support]
- Example: [specific example]

**Conclusion** (100 words)
- Summary: [key takeaways]
- CTA: [what to do next]
```

### Visual Requirements
```markdown
### Images Needed
- Hero image: [description]
- [X] screenshots: [of what]
- [X] diagrams: [showing what]
- [X] examples: [of what]

### Formatting
- Use subheadings every 300 words
- Include bullet lists for scanability
- Add pull quotes for emphasis
- Code blocks for technical content
```

### Competitive Analysis
```markdown
### Top Ranking Content
1. [URL] - [what they do well] - [gap we can fill]
2. [URL] - [what they do well] - [gap we can fill]
3. [URL] - [what they do well] - [gap we can fill]

### Our Angle
[How we'll differentiate/add more value]
```
